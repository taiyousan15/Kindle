## エグゼクティブサマリー

**目標**: 20分動画1本を100円前後、月100本で約1万円のコストで高品質な日本語リップシンク動画を作成する仕組みを構築する。

**結論**: **B案（RTX 4090 + VOICEVOX + MuseTalk）** が最もバランスが良く、目標を達成できる。クラウドGPUを使う **C案（Vast.ai + LatentSync）** なら25円/本と超低コストも実現可能。

| 案 | 構成 | コスト/本 | 月100本 | 目標達成 |
|---|---|---|---|---|
| **A案** | HeyGen SaaS | 224円 | 22,350円 | ❌ |
| **B案** | RTX 4090 + VOICEVOX | 105円 | 10,478円 | ✅ |
| **C案** | Vast.ai A40 + OSS | 25円 | 2,500円 | ✅ ⭐最安 |
| **D案** | ローカルGPU + OpenAI TTS | 118円 | 11,828円 | ✅ |

## 1. 現状分析：なぜ1本4,000円かかるのか

現在の主なコスト内訳（推定）:
- **HeyGen Avatar IV**: $0.10/秒 × 1,200秒（20分）= $120 ≈ **18,000円**
- または **SynthesiaやD-ID**: 月額サブスク $30〜$89 + 分単位課金

**問題点**: 商用SaaSは高品質だが単価が高く、20分動画には向いていない。

## 2. 技術選定：主要OSSリップシンクエンジン比較

### 2-1. MuseTalk v1.5（推奨 ★★★★★）

| 項目 | 詳細 |
|---|---|
| 開発元 | Tencent |
| ライセンス | CC BY-NC 4.0（非商用）→ 商用ライセンス要相談 |
| 技術 | GAN + 拡散ハイブリッド |
| 処理速度 | RTX 4090で45〜60 FPS（リアルタイム超） |
| 品質 | 口の動き精度が最高レベル |
| 日本語対応 | TTSと組み合わせ可 |
| 必要GPU | 最低RTX 3080（8GB VRAM） |

**特徴**: 2025年に大幅アップデート。ComfyUIワークフロー対応で非エンジニアでも使いやすい。

### 2-2. LatentSync（推奨 ★★★★☆）

| 項目 | 詳細 |
|---|---|
| 開発元 | ByteDance |
| ライセンス | **Apache 2.0（商用利用完全無料）** |
| 技術 | Diffusionベース、TREPA手法 |
| 品質 | 時間的一貫性が高い（長尺動画に強い） |
| 特徴 | ComfyUI統合、Hugging Face対応 |
| 必要GPU | A100推奨（A40/3090でも動作） |

**特徴**: 商用利用可能で法的リスクなし。長尺（20分）動画に特に強い。

### 2-3. SadTalker

| 項目 | 詳細 |
|---|---|
| ライセンス | Apache 2.0 |
| 処理速度 | 低速（RTX 3090で実時間の10倍） |
| 品質 | 顔全体の動きは自然だが口の精度はMuseTalk以下 |
| Replicate単価 | $0.0061/秒 |

→ 20分動画だと遅すぎるため非推奨。

### 2-4. LivePortrait

| 項目 | 詳細 |
|---|---|
| ライセンス | Apache 2.0 |
| 特徴 | 表情アニメーション特化、リップシンクより顔全体の表現力 |
| 用途 | アバター動画（口の動きより感情表現重視） |

→ リップシンクより「アバター動画」向き。

### 2-5. EchoMimicV2（注目）

| 項目 | 詳細 |
|---|---|
| 開発元 | Ant Group（アリペイ親会社） |
| 特徴 | 音声駆動、上半身アニメーション対応 |
| GPU | A100推奨 |
| 日本語対応 | 音声ドリブンなので言語非依存 |

## 3. 日本語TTS（音声合成）選定

| TTS | 品質 | コスト | 商用利用 | 備考 |
|---|---|---|---|---|
| **Style-Bert-VITS2** | MOS 4.37（人間同等） | **無料** | 要確認 | Columbia大学論文で証明 |
| **VOICEVOX** | 自然 | **無料** | 条件付き可 | ずんだもん等、日本語特化 |
| COEIROINK | 高品質 | 無料 | 商用可 | 独自キャラ音声 |
| OpenAI TTS-1 | 高品質 | $15/100万文字 | 可 | 20分≒約12,000文字 → $0.18/本 |
| NIJIVOICE | 自然 | 月額制 | 可 | API提供あり |
| ElevenLabs | 最高品質 | $0.30/1000文字 | 可 | 高コスト |

**推奨**: **Style-Bert-VITS2**（無料・最高品質）or **VOICEVOX**（安定・商用可）

## 4. コスト詳細試算

### A案: HeyGen SaaS（現状維持）

```
HeyGen Creator Pro: $49/月（25クレジット）
追加クレジット: 100本分 × $2.5/本 = $250
合計: $299/月 ≈ 44,850円/月
1本あたり: 約449円
```
❌ 目標の4倍以上

### B案: セルフホスト（RunPod + MuseTalk + VOICEVOX）★推奨

```
前提: 20分動画 = 1,200秒の音声/映像処理

RunPodクラウドGPU:
  - RTX 4090: $0.74/hr
  - 20分動画の処理時間: 約20〜30分
  - GPU費用: $0.74 × 0.5hr = $0.37 ≈ 56円/本

TTS（VOICEVOX）: 無料
ストレージ（R2/S3）: 約5円/本
その他（転送・管理）: 約44円/本

合計: 約105円/本
月100本: 10,500円 ✅
```

### C案: Vast.ai + LatentSync（最安値） ⭐

```
Vast.ai A40（48GB VRAM）: $0.32/hr（スポット価格）

LatentSync処理:
  - 20分動画: 30〜40分処理
  - GPU費用: $0.32 × 0.6hr = $0.19 ≈ 29円/本

TTS（Style-Bert-VITS2）: 無料
ストレージ: 3円/本
合計: 約25円/本 ⭐
月100本: 2,500円 ✅✅
```

※ Vast.ai はスポット価格のため変動あり（最安$0.10/hrも可）

### D案: ローカルGPU購入 + OpenAI TTS

```
RTX 4090購入: 250,000円
  → 月償却（36ヶ月）: 6,944円/月
  → 100本あたりの償却: 69円/本

電気代: 400W × 0.5hr × 100本 × 27円/kWh = 540円/月

OpenAI TTS-1: $15/100万文字
  → 20分原稿 12,000文字 × $15/100万 = $0.18 ≈ 27円/本

合計/本: 69 + 5 + 27 + 17 = 118円/本
月100本: 11,800円 ✅
```

## 5. 推奨システム構成（B案詳細）

```
[テキスト原稿]
    ↓
[VOICEVOX / Style-Bert-VITS2]
    ↓ 音声ファイル（WAV）
[MuseTalk v1.5 または LatentSync]
    ↓ (クラウドGPU: RunPod/Vast.ai)
[リップシンク済み動画（MP4）]
    ↓
[FFmpeg 後処理]
    ↓
[完成動画]
```

### 必要なソフトウェアスタック

```bash
# 環境
Python 3.10+
CUDA 12.1+
FFmpeg

# コアモデル
MuseTalk v1.5: https://github.com/TMElyralab/MuseTalk
LatentSync: https://github.com/bytedance/LatentSync

# TTS
VOICEVOX: https://voicevox.hiroshiba.jp/
Style-Bert-VITS2: https://github.com/litagin02/Style-Bert-VITS2

# ワークフロー自動化
ComfyUI（推奨）
または n8n / Make でバッチ処理
```

## 10. 最終推奨

### 即座に始めるなら: **B案（RunPod + MuseTalk + VOICEVOX）**
- 初期投資: なし（クラウドGPU使用）
- 月コスト: 約10,500円（100本）
- 構築期間: 約2週間
- 商用利用: MuseTalkのCC BY-NCに注意（非商用または要ライセンス取得）

### 完全商用・超低コストなら: **C案（Vast.ai + LatentSync + Style-Bert-VITS2）**
- 初期投資: なし
- 月コスト: 約2,500円（100本）
- ライセンス: Apache 2.0で完全商用OK
- 構築期間: 約2〜3週間
- 注意: Vast.ai のスポット価格変動に注意

### 長期的な自社運用なら: **D案（RTX 4090購入）**
- 初期投資: 250,000円
- 月コスト: 約11,800円（100本）
- 損益分岐: 購入後約18ヶ月でA案より安くなる
- 安定性: 最高（クラウド依存なし）
