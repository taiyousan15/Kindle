# 3. リサーチスキル一覧と動作フロー

## スキル全種別マップ

```
リサーチ指示
    │
    ├─ 速度優先
    │   ├─ research-free    → APIキー不要、Claude組み込み検索のみ
    │   └─ research         → Tavily + Brave + WebSearch（30秒）
    │
    ├─ 深さ優先
    │   ├─ deep-research     → 特定テーマ深掘り（Perplexity Sonar Pro）
    │   ├─ mega-research     → 複数ソース統合（Tavily+Brave+Perplexity+Twitter）
    │   └─ mega-research-plus → 8ソース完全並列（最大50件・最強）
    │
    ├─ 専門特化
    │   ├─ world-research    → 国際情報・経済・投資向け
    │   └─ intelligence-research → 経済指標・著名人発言・GIS 31ソース収集
    │
    └─ SNS特化
        └─ mega-research-plus（socialモード）→ Twitter+Reddit+Brave
```

## スキル別詳細仕様

- **発動条件**: 「調べて」「リサーチして」「確認して」
- **使用ソース**: Tavily → Brave → WebSearch（順次）
- **出力**: 要約レポート + 引用URL
- **コスト**: 約$0.01-0.05/回

### `mega-research-plus`（最強・8ソース並列）

```
│  Tavily + Brave + WebSearch → 15件、2-3秒            │
├─────────────────────────────────────────────────────┤
│ news（ニュース特化）                                  │
```

### `world-research`（投資・国際情報）
- **発動条件**: 「投資」「市場分析」「経済」「株」「暗号通貨」
- **使用ソース**: Alpha Vantage MCP + Exa MCP + Perplexity Deep Research
- **出力**: 金融データ付き詳細レポート
- **コスト**: 約$0.50-2.00/回

### `intelligence-research`（GIS 31ソース）
- **発動条件**: ディープリサーチと並行して自動稼働
- **収集対象**: 経済指標・著名人発言・ニュース
- **必須ルール**: mega-research, world-research実行時に必ず並行稼働

# 10. トレンド検知アルゴリズム

## TrendScore計算式

```
TrendScore = (
  0.40 × normalized(stars_7d_delta / total_stars)  // 急増率
+ 0.30 × normalized(npm_28d_growth_rate)            // DL成長率
+ 0.20 × normalized(hn_score + reddit_score)        // コミュニティ言及
+ 0.10 × recency_boost                              // 新着ブースト（30日以内+0.1）
)
```

## セマンティック検索設計

- **Embedding**: all-MiniLM-L6-v2（384次元・無料・ローカル実行）
- **Vector DB**: pgvector（MVP） → Qdrant/Weaviate（スケール時）
- **パイプライン**:
  ```
  クエリ → Embedding生成 → HNSW ANN検索
  → TrendScore重み付けリランキング → 上位20件返却
  ```

## アラート設計

| 条件 | アクション |
